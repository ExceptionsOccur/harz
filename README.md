# 中燃燃气查询 (hazr)

Home Assistant 自定义集成，用于查询中燃（中国燃气）燃气数据。
> 本项目由 D 指导鼎力支持

## 功能

- 查询燃气余额
- 查询本月用气量、本月账单
- 查询缴费记录（近6个月）
- 查询月度用气明细
- 查询用气地址、燃气户号
- 每6小时自动更新数据

## 传感器

| 实体 | 说明 | 单位 | 状态类 |
|---|---|---|---|
| `gas_balance` | 燃气余额 | CNY | total |
| `gas_last_month_usage` | 上月用气量 | m³ | total_increasing |
| `gas_last_month_payment` | 上月燃气费用 | CNY | total |
| `gas_last_payment` | 上次缴费金额 | CNY | total |
| `gas_last_payment_history` | 缴费记录列表 | - | - |
| `gas_address` | 用气地址 | - | - |
| `gas_account_no` | 燃气户号 | - | - |
| `gas_monthly_detail` | 月度用气明细 | - | - |

## 安装

### HACS 安装（推荐）

1. 在 HACS 中点击右上角菜单 → **自定义存储库**
2. 仓库地址：`https://github.com/ExceptionsOccur/hazr`
3. 类别选择：**集成**
4. 点击添加后，搜索 "中燃燃气查询" 并安装
5. 重启 Home Assistant

### 手动安装

1. 将 `hazr/` 目录复制到 HA 的 `custom_components/` 目录下
2. 重启 Home Assistant

## 配置

通过 HA 的"集成"页面添加：

1. **集成** → **添加集成** → 搜索 **中燃燃气查询**
2. 输入手机号和开户人姓名
3. 输入图形验证码
4. 输入手机短信验证码
5. 选择要绑定的燃气账户

## 技术说明

- 数据来源于中燃官方平台 `zrds.95007.com`
- 使用 `cloud_polling` 模式，每 6 小时轮询一次
- 支持重新认证（令牌过期后自动提示）
- 需 Home Assistant 2024.6 及以上版本
